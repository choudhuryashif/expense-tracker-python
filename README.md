# Expense Tracker

A simple Expense Tracker application built using Python. Users can add expenses, view all expenses, and calculate the total amount spent.

## Features

- Add Expense
- View Expenses
- Show Total Expense
- Exit Program

## Technologies Used

- Python
- Lists
- Loops
- Conditional Statements
- Functions

## How to Run

1. Install Python
2. Open terminal in the project folder
3. Run:

```bash
python expense_tracker.py
```
